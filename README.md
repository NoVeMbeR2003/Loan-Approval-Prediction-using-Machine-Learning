# Loan Approval Prediction using Machine Learning

## 📖 Description

This project predicts whether a loan application will be approved or rejected using a machine learning model. It uses a Decision Tree Classifier trained on a real dataset and allows user input for real-time prediction.

## 🚀 Features

* Data preprocessing and cleaning
* Label encoding of categorical data
* Model training using Decision Tree
* User input-based prediction system

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python loan_approval.py
